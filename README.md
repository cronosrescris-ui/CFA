## About

CFA (Continuous Flow Architecture) is a hydraulic continuous flow architecture that completely replaces the crankshaft. It is not an optimization. It is a replacement.

The architecture uses two inverted cones (suction + discharge) mounted coaxially, which geometrically eliminate dead points and mechanical friction. Energy is stored in hydraulic accumulators, and combustion is decoupled from mechanical output, allowing full fuel flexibility (hydrogen, ammonia, biofuels, diesel, gasoline).

---

## Advantages over classical crankshaft

| Feature | Crankshaft | CFA |
|---|---|---|
| Dead points | YES | NO |
| Systemic mechanical friction | YES | NO |
| Energy storage | NO (flywheel only) | YES (accumulator) |
| Fuel flexibility | Low | High |
| Dynamic load adaptation | NO | YES |
